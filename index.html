The Course Registration project with java


This Project is intended to create an user Friendly interface for the course Registration of students for an Engineering University

Advantages of this project

  • Time effiency
  • Database Connectivity
  • GUI with Java swing gives easy understanding and working
  • Robust code



    • This code is made with Java Swing

  • It's databases are accessed using a secure java jdbc connection
  • Database used : MYSQL Version 8.033
  • Inbuilt platform : Microsoft Windows 11
  • Project Status : Work in progress
